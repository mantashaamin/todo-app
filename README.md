## 📌 Overview

This **Productivity Dashboard** was built during my internship at **Cognifyz Technologies**.

Instead of a predefined project, problem statements were provided.  
Based on those requirements, I designed and implemented a solution combining:

- 📝 Task Management (To-Do App)  
- 🌦️ Real-Time Weather Tracking  

---

## ✨ Features

### 📝 To-Do App
✔ Add tasks  
✔ Delete tasks  
✔ Mark tasks as completed  
✔ Filter tasks (All / Active / Completed)  
✔ Persistent storage using Local Storage  

---

### 🌦️ Weather App
✔ Search any city  
✔ Real-time weather data  
✔ Displays:
- 🌡 Temperature  
- ☁️ Condition  
- 💧 Humidity  
- 🌬 Wind Speed  
- 🕒 Local Time
